퀵알람 사용하기

퀵 알람은 짧은 시간 뒤 빠르게 알람을 맞추고 싶을 때 사용할 수 있는 기능입니다.

복잡한 설정 없이 몇 번의 터치만으로 바로 알람을 등록할 수 있어 편리합니다.

사용 방법

메인 화면에서 하단의 + 버튼을 눌러 퀵알람을 빠르게 추가할 수 있습니다.

  1. 퀵 알람 창에서 원하는 시간과 간단한 옵션을 설정합니다.
  2. 체크 버튼을 누르면 바로 알람이 등록됩니다.

이런 경우에 유용해요

  • 잠깐 낮잠을 잘 때
  • 짧은 휴식 시간을 맞출 때
  • 빠르게 알림이 필요할 때

참고해주세요

퀵 알람은 빠른 설정이 장점인 기능입니다.

세부 설정이 필요한 경우에는 일반 알람 기능을 함께 사용해주세요.
